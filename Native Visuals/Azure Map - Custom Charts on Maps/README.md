# Azure Maps – Custom SVG Charts

<img width="1144" height="800" alt="Image" src="https://github.com/user-attachments/assets/d530639c-8b35-4e42-babe-1c77be3bb2fc" />

This example demonstrates how to use custom SVG images as markers in the Azure Maps visual in Power BI.
Instead of using the default map markers, you can generate SVG images dynamically with DAX and display custom visuals directly on the map, such as:

- ⭐ Star ratings
- 📊 Progress bars
- 🥧 Pie charts
- 📈 Mini charts
- 🚦 Status indicators
- 🎯 And many other SVG-based visuals

Because the SVG is generated dynamically, you can control its appearance using your data, including colors, values, icons, and labels.

## Included Examples
- Progress Bar
- Star Rating

These examples can be used as a starting point for building your own custom map markers.
