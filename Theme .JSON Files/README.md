## How to Use Power BI Themes

1. Download the desired `.json` file
2. In Power BI Desktop:
   - Go to **View** → **Themes** → **Browse for themes**
   - Select the JSON file
   - Apply the theme to your report

## Tips

- **Accessibility**: Many of these themes are designed with color contrast in mind.
- **Customization**: You can open any JSON file in a text editor to tweak colors, fonts, or data colors to better suit your brand.
- **Best Practices**: Use consistent themes across all reports in a project or organization.
