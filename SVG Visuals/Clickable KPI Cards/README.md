# SVG Button Slicer KPI Cards

A Power BI template showcasing high-density KPI cards built entirely with native Button Slicers and SVG backgrounds—no custom visuals required.

<img width="983" height="794" alt="Image" src="https://github.com/user-attachments/assets/c0db20a8-1d1e-48dd-b04f-c132a0d5f18d" />


##  ✨ Features
- Clickable KPI Cards – Filter your dashboard by Net Sales, Units, or Returns.
- Variance Display – Shows Actual, Previous Year, $ Variance, and Var % per tile.
- 100% Native – Uses only built-in Power BI visuals + inline SVG.
