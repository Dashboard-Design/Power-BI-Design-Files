# Custom Visuals

This folder contains a curated collection of popular Power BI custom visuals for reference and quick access.

The goal is not to archive every custom visual available for Power BI. Instead, this collection focuses on visuals that are widely used by the Power BI community, actively maintained, or provide capabilities beyond native visuals.

## These files can help you:

- Explore popular Power BI custom visuals
- Quickly install visuals without searching the marketplace
- Reproduce dashboards included in this repository
- Learn when and where different custom visuals are useful

## How to Use

### Option 1 – Import the .pbiviz file
1. Open Power BI Desktop.
2. In the Visualizations pane, select ... (More options).
3. Choose Import a visual from a file.
4. Select the desired .pbiviz file from this folder.
5. The visual will be added to your report and can be reused like any other visual.

### Option 2 – Install from AppSource

Most visuals included here are also available through Microsoft AppSource.

If you prefer, you can install the latest version directly from:

Visualizations → Get more visuals

## Version Notes

Custom visuals are updated by their authors over time.

The versions included in this repository are intended for convenience and reproducibility. If you need the latest features or bug fixes, consider downloading the newest version from Microsoft AppSource.

---

## Why This Folder Exists

Many reports in this repository use custom visuals that are not included with Power BI Desktop by default.

Providing the .pbiviz files here makes it easier to:

- Recreate dashboards
- Explore new visualization techniques
- Learn from existing report examples
- Save time searching for required visuals
